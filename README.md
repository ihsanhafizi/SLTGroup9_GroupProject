# SLTGroup9_GroupProject
System Thinking & Logic Assignment
This repository contains our group assignment for the System Thinking & Logic subject. The project focuses on identifying a real-world problem, analyzing the system,and proposing a logical solution using system thinking and logical reasoning principles.

This group assignment focuses on applying system thinking and logical problem-solving. The assignment involves analyzing a system's structure, breaking it into functional components, and representing its processes using flowcharts and pseudocode.
